PASOS PARA APRENDER

  • Elegir el tema a aprender
  • Investigar como me siento mejor para aprender
    • De forma visual
    • De forma auditiva
    • De forma kinestesica
  • Repasar lecciones:
    • Repatirlo conmigo mismo en pocas palabra
    • Decirle a otros y saber si queda claro
    • Haciendo preguntas y pruebas
      • Se vale equivocarse, pero se debe conocer por que
      • Prueba ser curioso y saber si existe solo una forma o una manera de ser
  • Anota o escribe, almacena la nueva informacion
  • Repite todo el proceso para nuevas lecciones
